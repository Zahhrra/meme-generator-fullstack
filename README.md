# Meme Generator Fullstack App

A fullstack meme generator built with:

- Node.js
- Express
- TypeScript
- Vanilla JavaScript
- Tailwind CSS

## Features

- Dynamic template loading
- Real-time meme preview
- Text positioning (x%, y%)
- Font & color customization
- PNG download support
- Backend validation with Zod
- Logging with Pino

## Run Locally

cd backend
npm install
npm run dev

Then open:
http://localhost:3000
